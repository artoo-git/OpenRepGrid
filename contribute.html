You can contribute!

Make suggestions

In order to make OpenRepGrid suit your needs we need to know what features you need, what you miss and what you feel needs improvement. So please don’t hesitate to tell us.

Build documentation

If you feel the documentation needs improvement or you have a use case you would like to see in the documentation, just get in touch. Or if you have a draft you would like to add, send it to us.

Supply code snippets

If have some basic R knowledge and come up with some code snippets just send them and we will take care to integrate your ideas into the package.

Join the dev team

If you have solid R knowledge just hammer out some code and send it to us :) Or, if you are familiar with git, fork the OpenRepGrid repo, make some changes and submit a pull request. Please also report bugs and suggest enhancements here.

Get in touch

  • Write an email to heckmann (dot) mark (at) gmail (dot) com.
  • There is a shared Dropbox folder for contributors. Have a look at it. If you want to get access to it e.g. to add some stuff, write an email.
  • (Expert level): Post a bug, issue or enhancement directly on github.
