Join in!

There are several ways of how to participate in the project.

Make suggestions

  • In order to make OpenRepGrid suit your needs we need to know what features you need, what you miss and what you feel needs improvement. So please donâ€™t hesitate to tell us. Just share your ideas in the comment section at the bottom of each page.

Build the documentation

  • If you feel the documentation needs improvement or you have a use case you would like to see in the documentation, just tell us in the comment section. Or if you have a draft you would like to add, send it to us.

Supply code snippets

  • If have some basic R knowledge and come up with some code snippets just send them and we will take care to integrate your ideas into the package.

Join us on github

  • If you have solid R knowledge just hammer out some code and send it to us :) Or, if you are familiar with git, fork the OpenRepGrid repo, make some changes and submit a pull request. Please also report bugs and suggest enhancements here.

Get in touch

You can get in touch in several ways:

  • Post your ideas, remarks and suggestions in the comment section at the bottom of each page. We will get back to your comments ASAP.
  • Write an email to openrepgrid@gmail.com
  • (Expert level): Post a bug, issue or enhancement directly on github

Contributors

The following people have contributed to the development of OpenRepGrid in one of the above ways. Become one of them!
